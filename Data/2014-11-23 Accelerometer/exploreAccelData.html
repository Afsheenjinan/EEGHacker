Exploring Accelerometer Data from OpenBCI¶

EEG Hacker, 2014-11-23, MIT License

Purpose¶

Record accelerometer data from OpenBCI V3 board during known motions. Confirm that the data matches the known motions.

Setup¶

OpenBCI V3 Board (see below) with software as delivered from OpenBCI. Data was logged using the OpenBCI GUI in Processing.

Procedure¶

I inserted the batteries to my OpenBCI board to give it power. I started the OpneBCI GUI to begin recording data. Holding the board in my hand, I completed the following manuevers:

  1. Start with board flat and level (z-axis points up)
  2. Roll it 90 deg to the right (x-axis points down) and 90 deg left (x-axis points up)
  3. Tip it nose down (y-axis points down) and nose up (y-axis points up)
  4. Flip it upside down (z-axis points down)

Notice the markings on the OpenBCI board that indicate the direction of the axes.

Data¶

The 3-axis accelerometer data was saved to a text file by the Processing GUI. I believe that the units in the log file are millimeters per second squared (mm/s2). I'll convert them from mm/s2 to m/s2 and then simply to G. I expect that the magnitude of the 3-axis acceleration vector should be close to 1.0 G most of the time.

Data Analysis¶

The data are processed here in Python. The goal is to confirm that the data is well behaved, that the units are correct, that the scale factors are correct, and that the correct axes are responding to the known motions.

# parse the data
# eeg_data_uV = data[:, 1:(8+1)] # EEG data, microvolts
accel_data_mm_s2 = data[:, 9:(11+1)]  # accel daa, mm/s2
accel_data_G = accel_data_mm_s2/1000.0/9.806  # convert to G

# create a vector with the time of each sample
t_sec = np.arange(len(accel_data_G[:, 0])) / fs_Hz

# The accel data seems to be padded with lots of zeros. 
I = np.nonzero(accel_data_G[:,0])
I=I[0]
print("Accel data exists in " + str(len(I)) + 
      " of " + str(len(t_sec)) + \
      " (" + str((100*len(I))/len(t_sec)) + 
      "%) of data packets");

# Only get the non-zero entries
accel_data_G = accel_data_G[I,:]
t_sec = t_sec[I]
