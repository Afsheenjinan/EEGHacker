
This is my repository holding software and data related to my EEG Hacker blog (http://eeghacker.blogspot.com).  I've organized this repository as follows:

* Arduino: This directory holds Arduino code for performing certain duties that are not tied to one specific EEG Hacker post.

* Matlab: This directory holds Matlab code not tied to one specific EEG Hacker post

* Data: This directory holds EEG data and analysis code that *is* tied to specific EEG Hacker posts
